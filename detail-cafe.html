Web Developer

Okt 2023

Objective

Creating a company profile website for Smooth Caffe that communicates information about the business in a simple and easily understandable way.

Tools and Tecknologi

PHP, HTML, Bootstrap, CSS and JavaScript

Challenge

The main challenge in building this website is to create a design that is simple and straightforward, yet still provides information that aligns with the café's concept.

The aim of this website is to deliver a simple and straightforward design that aligns with the café's concept while keeping visitors engaged. The design is crafted to ensure a user-friendly experience, featuring a clean layout and relevant content. This website is also designed to offer clear information about the café, including its menu. With this approach, visitors should gain a clear understanding of what the café has to offer and be inspired to visit.
