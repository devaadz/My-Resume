Objective

This project is a research requirement to make Classification of Eye Diseases.

Tools and Tecknologi

Python, Keras, Tensorflow, Bootstrap CSS

Challenge

The primary objective of this project is to develop a system in classifying images of Diabetic Retinopathy eye disease. Utilizing Python alongside Convolutional Neural Networks (CNNs) as the primary methodology, incorporating the Xception architecture

Some features in this application including:

  • Upload Image
  • Predicting Images According to Their Respective Classes
  • Responsoif Desain

The application utilizes Convolutional Neural Networks (CNNs) as the primary method for classifying Diabetic Retinopathy eye disease using images. In the process, the application determines the classes of these images using the Xception architecture. Some of the classes identified by the system include: No Diabetic Retinopathy, Mild Diabetic Retinopathy, and Moderate Diabetic Retinopathy.
