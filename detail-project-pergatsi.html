About Client

Company: PERGATSI (Persatuan Gateball Seluruh Indonesia)

Services: Backend Developer (Internship)

Objective

This project aims to create a backend utilizing Rest API to facilitate the transfer of data from the PERGATSI Sleman website to the PERGATSI DIY website.

Tools and Tecknologi

PHP, Laravel, Bootstrap, Vanilla Javascript, Postgree SQL

Challenge

To complete my internship assignment at university, I worked at PERGATSI, an athletic organization in the Yogyakarta area. My role was as a backend developer tasked with processing data transferred from the PERGATSI Sleman website to the PERGATSI DIY website using REST API and ensuring that the data is received by PERGATSI DIY.

I utilized Laravel as the framework to develop the PERGATSI Sleman website and seamlessly integrate it with the PERGATSI DIY website using Rest API. This facilitates PERGATSI DIY in showcasing athlete, referee, and player data from PERGATSI Sleman and implemented CMS system to manage and perform CRUD operations on PERGATSI member data for both websites. The website will feature functionalities including data filtering, searching, printing, and ect.
